# SillyTavern 叙事创作预设

用于 SillyTavern 的 Prompt 预设，专为**严肃向宏观叙事创作**设计。

## 文件说明

### A_test.json（推荐）
DefaultV4 的升级版本，采用全新的模块化架构，优化了文风读取机制，预留了 clewd 正则槽位。

**架构改进：**
- **模块化设计**：将所有规则拆分为独立模块（自然公理、认知边界、经济资源、反刻板、情绪控制、反机械化、防抢话等），通过 `setvar/getvar` 变量系统进行组装，便于单独启用/禁用
- **变量初始化机制**：统一的 `preset-init` 槽位预初始化所有变量，模块间解耦更彻底
- **分离式文风系统**：叙事框架（`fiction_style`）与文字规范（`writing_style`）独立为两个模块，分别通过变量注入，支持灵活替换
- **结构化思维链**：思维链内容由各模块动态注入检查项（`think_logic`、`think_scope`、`think_resource`、`think_style`、`think_user`、`think_output`），而非硬编码

**核心功能模块：**
- **自然公理**：因果闭环、资源守恒、组织摩擦、路径依赖、时间滞后、世界平行发展、随机性、现实边界、不确定性处理
- **认知边界**：信息有限、认知有限、理性有限——角色只能基于自身处境中可获得的信息行动
- **经济资源**：账目闭环、资金隔离、数字可追溯、资产不静默消失
- **反刻板**：规避集体单一意志、结果瞬时显现、用未来目光看当下、按身份标签行动
- **情绪控制**：角色情绪保持生活化克制，不以戏剧化夸大破坏性格底色
- **反机械化**：禁止 AI 式的条目列举和框架总结，角色对话须符合所处语境
- **防抢话**：不捏造用户角色的内心独白，不替用户做决策

**叙事与文字规范：**
- 现实主义叙事框架：冷静、克制、明确的基调，面向具备历史常识和社会经验的读者
- 第三人称外部叙述者，立场稳定；可进入人物意识但不赋予超时代知识
- 用词精准、中短句为主、白描优先，禁止空泛抒情和修辞堆砌
- 对话反映身份/教育/职业/地域差异，避免所有人一个腔调

**扩展槽位：**
- clewd/Regex 槽位、SPreset 配置槽位、作者槽位、额外信息/约束槽位、自定义任务槽位、摘要模块——均默认关闭，按需启用
- 内置正则脚本：`杀thinking`（前端隐藏思维链）、`杀用户输入`（深度>=3 时清除用户历史输入）

**默认模型配置：** `claude-opus-4-6`，200万上下文，64k 输出

---

### DefaultV4.json
A_test 的前身版本，核心理念相同但采用较早期的架构。核心功能：

- **现实主义叙事框架**：冷静克制的文风，强调因果闭环、信息有限、过程细节
- **防全知**：角色只能知道其处境中可获得的信息，禁止上帝视角
- **防刻板**：规避"集体单一意志"、"结果瞬时显现"等常见写作套路
- **自然公理**：资源守恒、组织摩擦、时间滞后、随机性等现实约束
- **防机械化**：禁止 AI 式的条目列举和框架总结，要求角色有血有肉
- **经济约束**：资金/资源变动必须账目闭环，可追溯
- **情绪克制**：角色情绪保持生活化，避免戏剧性过激反应
- **思维链**：正式输出前先用 `<thinking>` 标签进行结构化思考

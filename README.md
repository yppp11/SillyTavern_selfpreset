# SillyTavern 叙事创作预设

两个用于 SillyTavern 的 Prompt 预设，专为**严肃向宏观叙事创作**设计。

## 文件说明

### DefaultV4.json
通用创作助手预设。核心功能：

- **现实主义叙事框架**：冷静克制的文风，强调因果闭环、信息有限、过程细节
- **防全知**：角色只能知道其处境中可获得的信息，禁止上帝视角
- **防刻板**：规避"集体单一意志"、"结果瞬时显现"等常见写作套路
- **自然公理**：资源守恒、组织摩擦、时间滞后、随机性等现实约束
- **防机械化**：禁止 AI 式的条目列举和框架总结，要求角色有血有肉
- **经济约束**：资金/资源变动必须账目闭环，可追溯
- **情绪克制**：角色情绪保持生活化，避免戏剧性过激反应
- **思维链**：正式输出前先用 `<thinking>` 标签进行结构化思考
